About This Project

The Currency Exchange Tracker is a lightweight web application designed to help users access real-time exchange rates and visualize historical currency trends. It was built as part of my final project to combine my background in Agricultural Economics and my current journey in Software Development.

The project demonstrates how technology can make financial information more accessible, especially for communities and individuals who need quick insights into currency values for trade, travel, or study.

About Me
